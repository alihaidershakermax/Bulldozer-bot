# 🚜 بوت بلدوزر - Bulldozer Bot

بوت تيليغرام متخصص في استخراج النصوص من PDF والصور وترجمتها إنجليزي → عربي عراقي.

## المميزات
- OCR باستخدام Tesseract + OpenCV
- ترجمة تلقائية (Bing / Google fallback)
- PDF ثنائي اللغة بتنسيق احترافي
- واجهة عربية عراقية كاملة
- لوحة تحكم للأدمن عبر Telegram
- قاعدة بيانات Convex
- نشر على Koyeb

## التثبيت
```bash
git clone https://github.com/alihaidershakermax/Bulldozer-bot
cd Bulldozer-bot
pip install -r requirements.txt
cp .env.example .env
python bot.py
```
